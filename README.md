📱 Jio Clone Website

A responsive Reliance Jio Website Clone built using HTML & CSS.
This project replicates the design, layout, and UI of the Jio official website, focusing on clean structure and responsiveness.

🌐 Live Demo

👉 https://jio-rabanadas-b37c64.netlify.app/

🏢 About Jio

Reliance Jio is one of India’s largest telecommunications companies, offering services like mobile networks, broadband, and digital solutions. It has over 500 million users and played a major role in making internet access affordable in India.
